# Finance Module Folder Structure

1. **supplier-ledger**: Supplier ka hisaab (Khaata).
2. **customer-ledger**: Customer ka hisaab.
3. **payment-entry**: ABC Enterprises ko paise dene ki entry.
4. **receipt-entry**: Customer se paise lene ki entry (GPay, Cash, etc).
5. **p-and-l-dashboard**: Profit and Loss report.
6. **reports**: Tax and other reports.
7. **models**: Data types for finance.
8. **service**: API communication.
